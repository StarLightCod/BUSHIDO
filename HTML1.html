Экипировка для Кендо

Богу (доспехи)

Защитное снаряжение, состоящее из маски (мен), нагрудника (до), перчаток (котэ) и защиты для бедер (тарэ).

Материал: Прочная ткань, бамбук, кожа
Вес: 5-7 кг
Обязательно: Да

Синай

Бамбуковый меч, состоящий из четырех бамбуковых полос, скрепленных кожей. Используется для тренировок и соревнований.

Длина: 120-122 см (для взрослых)
Вес: 400-550 г
Обязательно: Да

Кендоги

Традиционная одежда для кендо: куртка (кендоги) и широкие штаны (хакама). Обычно изготавливаются из хлопка или синтетических материалов.

Цвет: Индиго, черный, белый
Материал: Хлопок, полиэстер
Обязательно: Да

Экипировка для Иайдо

Иайто

Учебный меч с незаточенным лезвием из цинкового сплава. Используется для тренировок вместо настоящей катаны.

Длина: Зависит от роста практикующего
Вес: 700-1000 г
Обязательно: Да

Оби

Широкий пояс, который поддерживает ножны меча и фиксирует хакаму. Обычно изготавливается из прочной хлопковой ткани.

Ширина: 10-15 см
Длина: 3-4 м
Обязательно: Да

Хакама

Традиционные японские штаны с широкими складками. В иайдо используются более плотные и тяжелые хакамы, чем в кендо.

Материал: Хлопок, полиэстер, твил
Цвет: Черный, синий, белый
Обязательно: Да

Экипировка для Дзёдо

Дзё

Деревянная палка длиной около 128 см и диаметром 2,4-2,6 см. Изготавливается из прочных пород дерева (дуб, бук, граб).

Длина: 128 см (стандарт)
Вес: 450-600 г
Обязательно: Да

Боккэн

Деревянный меч, используемый для тренировок. В дзёдо применяется для имитации атак противника.

Длина: 102 см (стандарт)
Материал: Дуб, бук, граб
Обязательно: Да

Защитное снаряжение

Включает перчатки (котэ), защиту для запястий и иногда легкий нагрудник. Используется на продвинутых этапах тренировок.

Тип: Облегченный вариант
Материал: Кожа, пластик, ткань
Обязательно: Для продвинутых
